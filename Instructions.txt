How to use bot
NOTE: This bot is not tested for anything other then the Lingco vocab sets. It MAY NOT WORK on anything other then the vocab sets.

you will need 4 things. i recommend getting notepad up and copy-pasting into it


you will need the vocab set id, the jwt token, the session id, and the session uuid. here is how to get all of them

=========================

for the jwt token: 
1. go to lingco
2. hit F12, and open the network tab
3. do one question with the network tab open. 
4. click on the post request that is appears. 
5. click on the headers tab 
6. scroll until you find the line that says
	Authorization: Bearer
7. copy everything after the Bearer. (do not copy the bearer text. only everything after)
8. this is your jwt token. paste it in notepad and have it ready

Note: this resets often. every time you open up a new set, get a new token

=========================

for the session id:
1. in the same post request as step 1, under the headers tab, there will be a link. it should look like this: https://class.lingco.io/api/vocab_sets/sessions/14057596/responses
2. take the numbers after /sessions/
3. copy paste them into notepad

Note: this also changes every time you open a new session. grab a new one every time. 

=========================

for the session uuid:

1. In the same POST request from Step 1
2. In the Headers tab, find the line:
       X-User-Session-UUID: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
3. Copy the value after the colon
4. This also resets every set. 

=======================
for the set id:


1. Look at the URL in your browser while on the vocab set page
2. It will look like:
       https://class.lingco.io/content/vocab_sets/XXXXX/session/learn
3. The number where XXXXX is = your Vocab Set ID

======================
